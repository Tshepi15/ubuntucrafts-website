# Ubuntu Crafts Cooperative Website

E-commerce platform for handcrafted products from Limpopo Province artisans.

## About
Ubuntu Crafts Cooperative is a community-based enterprise producing handcrafted pottery, woven baskets, traditional clothing, and organic skincare items. Supporting rural economic development through authentic African crafts.

## Features
- Responsive design
- Handcrafted product showcase
- AWS API integration
- Secure e-commerce platform

## Technologies
- HTML5, CSS3, JavaScript
- AWS Lambda & API Gateway
- SVG product images

## Products
- Handcrafted Pottery
- Woven Baskets
- Traditional African Clothing
- Organic Skincare Items
